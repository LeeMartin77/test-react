# TEST 003 — Feature: User page linked from a post

Summary
-------

Add a simple user page and link to it from each individual post page using the post’s user id.

Acceptance criteria
--------------------------

1. A user page exists at an appropriate route and renders without error.
2. The individual post page includes a visible link to the user page for the post’s author.
3. The user page displays the user’s name.

Notes
-----

- Use the user id that is part of a post object to fetch the single-user data.
- Reference for the single-user API: https://dummyjson.com/docs/users#users-single
